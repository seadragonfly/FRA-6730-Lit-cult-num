## Disclaimer sur l’utilisation des LLM

ChatGPT (5.2) et Claude (Sonnet 4.5) ont été utilisés comme assistants dans l’écriture du code qui implémente le modèle formel. Plus précisément, ils ont servi à améliorer une première version du code et à proposer une pipeline pour les parties les plus complexes de l’implémentation, notamment celles concernant la transformation de données CSV en JSON selon au modèle proposé ([f-finals](./f-finals.ipynb) par exemple). Les LLMs n’ont contribué à aucune idée originale dans la conception du modèle, et le code a été vérifié afin de s’assurer qu’il implémente correctement la structure JSON souhaité.
